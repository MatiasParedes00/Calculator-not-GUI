Este es un primer experimento para agregar a mi portafolio de trabajo.
Lo que hice es recrear un mini-proyecto de programación de cuando estaba aprendiendo, con mejoras significativas en cuanto a legibilidad y estilo en relación a la versión anterior.

El propósito de este proyecto es crear una calculadora científica en el Shell de Python. Funciona de la siguiente manera:
1) El usuario ingresa un número del 1 al 3, cada una relacionada con una acción específica. Si el usuario ingresa 1, va a realizar una operación de un sólo operando (seno, coseno, etc., que normalmente sólo admiten una variable de entrada). Si el usuario ingresa 2, será una operación de dos operandos (suma, resta, etc., que normalmente admiten dos variables de entrada). Si el usuario ingresa 3 se le preguntará si realmente quiere salir, y en caso afirmativo (O sea, si el usuario ingresa "s") saldrá de Python, pero si el usuario ingresa "n" el programa seguirá normalmente.
2) Luego, el usuario será preguntado qué operación específica de la lista quiere realizar, de una lista que dependerá de la elección en el paso 1). Podrá elegir esa operación ingresando el número apropiado.
3) Finalmente, se le pedirá al usuario que ingrese la(s) variable(s) que participa(n) de la operación, para luego mostrarse en pantalla el resultado y seguir con el programa.

Como verán, este programa es más bien básico, pero como dije, este era un mini-proyecto para aprender programación. A parte, vale la pena aclarar que esto también es para aprender a usar Git.

Como última aclaración, el nombre de los archivos está en Inglés, pero el programa está en Español. Esto es tanto por comodidad mía (Trabajo mejor escribiendo y leyendo en inglés) como para el potencial contratador (Que posiblemente tenga mejormanejo de Español que de Inglés)

This is my first experiment to add to my work portafolio.
What I've done here is to recreate a mini-project I did when learning how to program, with great improvements in matter of legibility and style compared to the past version.

The propouse of this proyect is to make a Shell-based scientific calculator. Works in this way:
1) The user enter a number from 1 to 3, each number related to an specific action. If the user enters a 1, they are gonna make a single input operation (functions like ln(a), sin(a), etc., which usually requieres just a single input). If the user enters a 2, they are gonna make a doble input operation (functions like a + b, a / b, etc., which usually requieres two inputs). If the user enters a 3, they are promted if they are sure to exit the aplication, and if they enters a "s" (shorthand for "Si", yes in Spanish) the user exits, and if the user enters "n" the program continues normally
2) Then, the user is shown a list of possible operations based on the choice in step 1). The user can choose entering the appropiate number.
3) Finally, the user will be asked to enter the variable(s) that are part of the operation, and the result will be printed. Then, the program will run normally

As you can see, this program is really basic, but as I said, this is mini-project to learn coding. Also, it worth mentioning that this is also to learn to use Git.

As a last aclaration, the file's name are in English, but the program is in Spanish. This is for both my comfort (I work better reading and reading in English) and my potential recruiter's comfort (They will probably more comfortable with Spanish than with English)
